This directory contains the image assets needed for the Filipino healthcare workers nonprofit website.

- `logo.png` - Main logo (recommended: 200x80px)
- `logo-white.png` - White version of logo for dark backgrounds
- `favicon.ico` - Website favicon

### Hero Section

## Image Guidelines:

### Style and Content
- Professional, warm, and welcoming tone
- High-quality, well-lit photographs
- Diverse representation of Filipino healthcare workers
- Mix of individual portraits and group shots
- Healthcare settings (hospitals, clinics, care facilities)

### Technical Specifications
- Format: JPG for photos, PNG for logos/graphics
- Resolution: Minimum 1920px wide for hero images
- Optimization: Compress for web (under 500KB for large images)
- Alt text friendly: Clear, descriptive subjects

### Cultural Sensitivity
- Respectful representation of Filipino culture
- Professional healthcare attire
- Positive, empowering imagery
- Authentic Filipino healthcare professionals when possible

### Suggested Photo Sources
1. Professional photographer specializing in healthcare
2. Stock photo sites with Filipino/Asian healthcare worker collections
3. Partnerships with Filipino healthcare organizations
4. Original photography from community events and programs

### Color Palette for Graphics
- Primary Blue: #0038A8 (Philippine flag blue)
- Primary Red: #CE1126 (Philippine flag red)  
- Primary Yellow: #FCD116 (Philippine flag yellow)
- Supporting colors: Ocean blue, warm whites, professional grays

## Usage Rights
Ensure all images have proper licensing for commercial/nonprofit use and respect copyright and model release requirements.
