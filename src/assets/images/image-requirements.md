## Technical Specifications

### File Formats
- **Logos**: PNG with transparent background
- **Photos**: JPG (optimized for web)
- **Icons**: SVG when possible, PNG fallback

### Size Requirements
- **Hero images**: Max 1920px width, optimized under 500KB
- **Testimonials**: 150x150px, under 50KB each
- **Logos**: Vector format preferred, 200x80px display size

### Optimization
- Compress all images for web
- Use WebP format when supported
- Provide retina-ready versions (2x size)

## Cultural Considerations

### Design Elements

- Positive, empowering representation
