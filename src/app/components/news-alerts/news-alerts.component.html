How to Add New Facebook Posts

To add new Facebook posts to this section:

  1. Copy the Facebook post URL
  2. Add a new alert-item div in the news-alerts component
  3. Use the app-facebook-post component with the post URL
  4. Set appropriate title, description, and urgency level
<app-facebook-post
  postUrl="YOUR_FACEBOOK_POST_URL"
  title="Your Title"
  description="Your description">
</app-facebook-post>
